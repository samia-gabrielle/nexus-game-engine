## TECHNICAL SPECIFICATIONS

### File Format
- **Type:** JavaScript (ES5-compatible)
- **Exports:** CommonJS (Node.js) + Browser (window object)
- **Size:** 85 KB uncompressed
- **Syntax:** Valid JavaScript, no errors
- **Load Time:** <100ms on modern browsers

### Data Structure
```javascript
ACT3_SCENES = {
  "h1a_scene": { t, i, txt, o[], e, relationships, ... },
  ...
}

PARALLEL_REALITIES = {
  "p1a_scene": { t, i, txt, o[], e, relationships, ... },
  ...
}

MASSIVE_CADERNO = {
  "entry_id": { t: title, d: description },
  ...
}

MASSIVE_ECOS = {
  "milestone_id": { t: title, d: description },
  ...
}
```

### Scene Fields
- `t` - Scene title (Portuguese)
- `i` - Image reference
- `txt` - Narrative text (Portuguese)
- `o` - Choices array (choice text + next scene)
- `e` - Associated ECOS milestone
- `relationships` - NPC relationship changes
- `soundTheme` - Audio theme for scene
- `artifact` - Collectible item (optional)
- `achievement` - Achievement unlock (optional)
- `pathLock` - Path restriction (optional)
