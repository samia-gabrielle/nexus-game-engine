## 🎮 HOW TO IMPLEMENT

In `game.js`, update `getUnlockedEchoes()`:

```javascript
const ECHOES_DATA = {
  "awakening": {
    title: "The Waking",
    text: "Você reconhece a sensação de retorno..."
  },
  // ... etc for all 30
};

function getUnlockedEchoes(state) {
  return state.echoesUnlocked.map((id) => ECHOES_DATA[id] || { id, title: id });
}
```

This gives the Echoes panel real content instead of generic placeholders!
