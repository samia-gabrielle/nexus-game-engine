## CONTINGENCY PLANNING

### If Behind Schedule
1. Reduce scope to MVP (only 2150 + 1920)
2. Cut non-critical echoes (keep only 12 instead of 30)
3. Defer audio to later
4. Focus on narrative + systems before art

### If Narrative Quality Is Low
1. Pause implementation
2. Do narrative pass with external reader
3. Revise scenes
4. Restart implementation

### If Players Don't Engage with Ambiguity
1. Add more guidance on theme
2. Refine ending messaging
3. Create "director's commentary" explaining design

## SUCCESS CRITERIA (Final)

The game succeeds when:

1. ✅ **Central question is clear:** Player knows the game is about legacy, not timeline-fixing
2. ✅ **Echoes feel meaningful:** Not collectibles, but semantic fragments
3. ✅ **Ambiguity is comfortable:** Endings don't feel incomplete, just multivalent
4. ✅ **Characters are distinct:** Each NPC has a voice, not just an opinion
5. ✅ **Relationships matter:** High/low relationships create different paths
6. ✅ **Technical works:** No bugs, save/load reliable, dialogue parses
7. ✅ **Emotional impact:** Players report being moved
