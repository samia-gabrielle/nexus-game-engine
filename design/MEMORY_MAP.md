# NEXUS — Memory Unlock Map

**Purpose:** Define how Echoes combine to unlock Memories across the game

---

## HOW THE SYSTEM WORKS

### Single Echo
When you meet someone, you gain an echo—a fragment of who they are.

Example:
```
Echo: "THOMAS — The Liberator"
"A scientist who believed his work would free humanity."
```

### Combined Echoes
When you collect echoes from multiple scenes/eras, they combine to unlock deeper memories.

Example:
```
Echo: "THOMAS — The Liberator" (1920)
  +
Echo: "ALICE — The Revolutionary" (1970)
  =
MEMORY UNLOCKED: "The Cascade"
(A scene showing how Thomas's idea became Alice's revolution)
```

### Memory Function
A memory is a **cutscene or internal monologue** that reveals:
- How an idea traveled across time
- The moment a philosophy changed
- A connection between two people/eras
- A contradiction the player needs to understand

## CORE MEMORY UNLOCKS (MVP)

For the vertical slice (first 4 weeks), define 6 memories:

### Memory 1: "The Name"
**Unlock Requirement:**
- ECHO echo #1 (The Honest)
- RAIN echo #1 (The Believer)

**Scene:** Player sits with RAIN. They piece together fragments.

**Revelation:** Your name was never important. What matters is what people call you.

**Emotional Beat:** Acceptance that identity is relational, not fixed.

**Duration:** 2 minutes

---

### Memory 2: "The Tower"
**Unlock Requirement:**
- ARCHITECT echo (The Heir)
- RAIN echo #2 (The Guardian)

**Scene:** ARCHITECT shows you a video log from 2050 of you making a decision in THIS tower.

**Revelation:** You built the tower. Or you destroyed it. Depending on perspective.

**Emotional Beat:** Realization that the environment itself carries your history.

**Duration:** 3 minutes

---

### Memory 3: "The Cascade"
**Unlock Requirement:**
- THOMAS echo — The Liberator (1920)
- ALICE echo — The Revolutionary (1970)
- DIRECTOR echo — The Mirror (2150)

**Scene:** A montage showing how Thomas's single idea became Alice's movement became Director's corporation.

**Revelation:** One philosophy can become three contradictory outcomes depending on who holds it.

**Emotional Beat:** Understanding that impact is not predictable.

**Duration:** 4 minutes

---

### Memory 4: "The Choice"
**Unlock Requirement:**
- THE PIVOT echo (2050, literally yourself)
- MAYA echo — The Believer (2050)
- REVOLUTIONARY echo — The Test (2050)

**Scene:** Internal monologue where The Pivot (you, 20 years ahead) explains the decision you're about to make.

**Revelation:** You made the right choice with incomplete information. Or you made the wrong choice with the best intentions. The game doesn't judge.

**Emotional Beat:** Acceptance of moral ambiguity.

**Duration:** 3 minutes

---

### Memory 5: "The Path Not Taken"
**Unlock Requirement:**
- PROPHET echo — What You Could Have Been (1970)
- COLLECTIVE echo — The Dream (1970)
- SKEPTIC echo — The Honest (1970)

**Scene:** A glimpse into the life you could have lived if you chose 1970 over returning.

**Revelation:** Utopia has costs too. No choice is purely good.

**Emotional Beat:** Bittersweetness; understanding sacrifice.

**Duration:** 3 minutes

---

### Memory 6: "The Question"
**Unlock Requirement:**
- THOMAS echo — The Question (1920)
- MARIA echo — The Witness (1920)
- ECHO echo — The Ancient (constant)

**Scene:** The moment Thomas asks you "Did my work matter?" and you realize you can't answer.

**Revelation:** You can witness consequences, but you can't judge whether they were worth it.

**Emotional Beat:** Humility; the limits of understanding.

**Duration:** 4 minutes

## MEMORY DESIGN PRINCIPLES

### Principle 1: Memories Don't Explain
A memory should *complicate*, not clarify.

❌ Bad memory: Shows exactly what happened
✅ Good memory: Shows what happened AND reveals two contradictory ways to interpret it

### Principle 2: Memories Are Interactive
When possible, memories should include a *reflection choice*.

Example: After watching "The Cascade" memory, player is asked:
- "Do you accept responsibility for this cascade?"
- "Do you blame the people who twisted your idea?"
- "Do you accept that you can't control how ideas evolve?"

Each answer shifts slightly how the player is perceived.

### Principle 3: Memories Can Contradict
Two different memory unlocks can show the *same event* from different perspectives.

Example:
- Memory: "The Tower" (ARCHITECT's view) = "You built this temple to human connection"
- Memory: "The Cage" (DIRECTOR's view) = "You built this monument to control"
- Both true from different angles

### Principle 4: Some Memories Are Secret
New Game+ unlocks "hidden memories" that recontextualize everything.

Example: A scene where you see yourself making a choice in 2050, knowing now it was wrong—and choosing it anyway.

```json
      "unlocked": false,
      "choice_enabled": true,
      "affects_ending": "connector"
    }
  }

```

### Echo Combination Logic

```
When player gains an echo:
  1. Check if any memory requirements are met
  2. If memory_required_echoes ⊆ player_echoes:
     → UNLOCK MEMORY
     → Add to "available memories" menu
     → Show unlock notification

When player views memory:
  1. Play cutscene
  2. Show choice (if enabled)
  3. Record choice outcome
  4. Update ending calculations
```
