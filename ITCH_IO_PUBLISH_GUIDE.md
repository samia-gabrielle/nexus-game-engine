# Como Publicar NEXUS no itch.io

## PASSO 1: Criar Conta itch.io
1. Vá para https://itch.io
2. Clique "Register" (canto superior direito)
3. Crie conta com email whatsn@gmail.com
4. Confirme email

## PASSO 2: Preparar Arquivo do Jogo
- Arquivo: `nexus-complete.html` (46 KB)
- Localização: `c:\Users\54547130159\nexus\nexus-complete.html`
- Certifique-se que o arquivo está pronto para publicar

## PASSO 3: Criar Novo Projeto no itch.io
1. Acesse https://itch.io/dashboard
2. Clique "New Project"
3. **Preencha os campos:**
   - **Project title**: NEXUS
   - **Project URL**: nexus-game (URL única)
   - **Classification**: Game
   - **Kind of project**: HTML (no upload required)
   - **Description**: Cole o conteúdo de `ITCH_IO_DESCRIPTION.md`

## PASSO 4: Upload do Arquivo
1. Desça até "Uploads"
2. Clique "Upload new file"
3. Selecione `nexus-complete.html`
4. Deixe como **Playable in browser**
5. Clique "Save & publish"

## PASSO 5: Configurar Preço
1. Desça até "Pricing"
2. Selecione "Free" OU "Free with variable pricing"
3. Se variable pricing:
   - Minimum price: $0.00
   - Suggested price: $0.00
   - Permite "Pay What You Want"
4. Salve

## PASSO 6: Adicionar Screenshots/Imagens
1. Desça até "Gallery"
2. Clique "Add images"
3. Adicione screenshots do jogo:
   - Tela inicial
   - Exemplo de cena
   - Menu lateral (Caderno/Ecos)
   - Diferentes realidades se possível

## PASSO 7: Tags e Categorias
- Tags: `narrative`, `branching-narrative`, `sci-fi`, `story`, `indie-game`, `text-adventure`, `multidimensional`
- Category: Puzzle, Adventure, or Experimental

## PASSO 8: Publicar
1. Scroll até o final
2. Clique "Save" e depois "Publish"
3. Página fica live em: https://samia-gabrielle.itch.io/nexus-game
