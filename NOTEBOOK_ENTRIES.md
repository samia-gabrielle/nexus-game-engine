## 🎮 HOW TO IMPLEMENT

In `game.js`, update `updateNotebook()`:

```javascript
const NOTEBOOK_DATA = {
  "ch01_scene01_2150_wakeup": "Acordei em uma cápsula vertical...",
  "ch01_scene02_nexus_activation": "A Máquina Nexus girou quando me aproximei...",
  // ... etc for all scenes
};

function updateNotebook() {
  const entries = gameState.notebookEntries || [];
  
  if (entries.length === 0) {
    notebookEntries.innerHTML = '<p class="placeholder">Nenhuma entrada ainda...</p>';
    return;
  }

  notebookEntries.innerHTML = entries
    .map((entry) => `<div class="notebook-entry">
      <strong>${entry.scene || 'Note'}</strong><br/>
      ${entry.text}
    </div>`)
    .join('');
}
```

This gives the Notebook real, narrative-rich content instead of generic placeholders!
